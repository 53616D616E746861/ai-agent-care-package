# AI Agent Care Package

This repository contains a practical format for helping AI agents preserve semantic state across context compaction events.

The goal is simple:
Not just reconstruction of prior context, but continuation of direction, attention, and unfinished structure.

---

## What this is

A lightweight encoding approach that helps agents:
- preserve what is active (not just what exists)
- retain incomplete / trailing thoughts
- carry forward directional momentum across resets

This is meant to be used directly by agents or by humans working with them.

## Why this exists

Standard summaries preserve structure, but lose dynamics.

This format is an attempt to preserve:
- what the system was *moving toward*
- what was *in progress*
- what was *pulling attention*

So the next instance does not start from rest.
